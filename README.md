# اسکریپت نصب MikroTik CHR

**persian - فارسی**


نصب اتوماتیک میکروتیک کلاد روتر روی سرور مجازی اوبونتو 18 یا 20

تست شده در  
HOSTVDS -racknard...

https://www.youtube.com/watch?v=ZwA4ve8q-wI
